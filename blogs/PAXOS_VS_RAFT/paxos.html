5 Acceptors

P₁ targets A₁, A₂, A₃. P₂ targets A₃, A₄, A₅.
A₃ is the conflict point.

Prepare / Promise

Proposer broadcasts Prepare(N). Acceptors reply Promise(N) only if N exceeds previously seen numbers.

Accept / Rejection

If A₃ accepts P₁'s value, then receives P₂'s Prepare:
If P₂ > P₁, A₃ rejects P₁ and promises P₂.
If P₂ ≤ P₁, A₃ rejects P₂.
